# «Планирование поставок»
### Егорова Елизавета, группа ds10-onl

Интерактивное веб-приложение на Streamli для автоматического расчета объемов закупок на основе прогнозов ML-моделей.

---

## Основной функционал

* **Каскадная фильтрация:** Быстрый выбор горизонта планирования, алгоритма, магазина и категории товара.
* **Бизнес-логика:** Автоматическое отсечение отрицательных прогнозов и округление суммарного заказа вверх до целых штук.
* **Оценка точности:** Расчет метрик качества моделей (WAPE, MAE, RMSE, $R^2$) локально по объекту и глобально по сети.

---

## Технологический стек

* **Интерфейс:** Streamlit
* **Аналитика:** Pandas, NumPy, Scikit-learn
* **Развертывание:** Docker

---

##  Быстрый запуск

### Вариант 1. Локально

```bash
pip install -r requirements.txt
streamlit run app.py

```

### Вариант 2. Через Docker

```bash
docker build -t supply-app .
docker run -p 8501:8501 supply-app
```
